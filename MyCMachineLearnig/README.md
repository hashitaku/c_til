# MyMachineLearning

## ビルド
~~~
mkdir build
cd build
cmake ..
make
~~~
ビルドすると`build`フォルダではなく`CMakeLists.txt`と同じ階層に`./a.out`が作られます。

## データファイル

データファイルは`make_data`の中の`./a.out`を実行すると`main.c`と同じ階層に作られます。
